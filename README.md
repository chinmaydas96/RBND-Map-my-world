# Project4_Map_my_world

To build
```
cd catkin_ws
catkin_make
```
To run

Terminal 1
```
source devel/setup.bash
roslaunch my_robot world.launch 
```
Terminal 2
```
Make "teleop_twist_keyboard.py" executable.
source devel/setup.bash
rosrun teleop_twist_keyboard teleop.launch 
```
Terminal 3
```
source devel/setup.bash
roslaunch mapping mapping.launch
```
